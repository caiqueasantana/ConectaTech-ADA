Portfólio e Apresentações do Projeto ConectaTech ADA

Esta seção consolida os principais entregáveis e artefatos do Projeto ConectaTech ADA, servindo como um portfólio abrangente das competências desenvolvidas durante a monitoria. Ele demonstra minha capacidade de planejar, desenvolver e implementar soluções tecnológicas com impacto educacional e social.

1. Relatório Final do Projeto ConectaTech ADA

O documento detalhado que descreve todo o processo do projeto, desde a concepção até os resultados e lições aprendidas. Inclui informações técnicas e pedagógicas.

Baixar Relatório Final (PDF)

2. Repositório de Código-Fonte Completo

Acesse o código-fonte completo do Módulo de Software Multiplataforma e do Firmware do Módulo de Hardware Embarcado. O repositório inclui instruções para setup e execução.

Visitar Repositório GitHub

Tecnologias Envolvidas: Python (Multiplataforma), C/C++ (Embarcado), HTML/CSS/JavaScript (se aplicável para web-frontend), Microcontrolador ESP32.
