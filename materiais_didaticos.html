Materiais Didáticos e Oficinas

Esta seção detalha a metodologia de ensino desenvolvida para o "ConectaTech ADA" e os materiais criados para guiar os participantes em suas primeiras experiências com programação e automação, interligando o software multiplataforma com o hardware embarcado. Nosso objetivo é tornar o aprendizado acessível, prático e divertido.

Visão Geral da Metodologia Pedagógica

Adotamos uma abordagem *hands-on* e baseada em projetos, focando na resolução de problemas reais com o uso do Kit ConectaTech. A metodologia é iterativa, permitindo que os alunos construam conhecimentos gradualmente, desde a lógica básica até a interação com o hardware e a visualização de dados. Incentivamos a experimentação e a colaboração.

Roteiros de Aula e Módulos de Aprendizado

O conteúdo é dividido em módulos temáticos, cada um com objetivos claros, tópicos abordados e desafios práticos. Abaixo, exemplos dos módulos iniciais:

Módulo 1: Primeiros Passos na Lógica e LEDs (Introdução ao Software Multiplataforma)

  • Objetivos: Introduzir conceitos de Algoritmos, Sequência, Seleção e Repetição. Familiarizar com o ambiente de programação visual da plataforma.
  • Tópicos: O que é um algoritmo, comandos sequenciais, blocos de repetição (loops), blocos condicionais (se/então/senão).
  • Desafio Prático: "Pisca LED Virtual" - Criar um algoritmo na plataforma para simular o piscar de um LED virtual.
  • Recursos: Plataforma ConectaTech ADA (Módulo Software), Roteiro de Aula 1.

Módulo 2: O Código Ganha Vida! (Introdução ao Hardware Embarcado)

  • Objetivos: Conectar o software multiplataforma ao hardware embarcado. Entender a comunicação entre eles e controlar um LED físico.
  • Tópicos: Introdução ao microcontrolador ESP32 (ou similar), GPIOs, comunicação serial (UART), conceitos de entrada e saída digital.
  • Desafio Prático: "Controle de LED Físico" - Usar a plataforma para ligar e desligar um LED conectado ao Kit ConectaTech, enviando comandos.
  • Recursos: Kit ConectaTech ADA (Módulo Hardware e Software), Roteiro de Aula 2, Fritzing ou diagrama de montagem do LED.

Módulo 3: Sensores e Dados (Interação e Visualização)

  • Objetivos: Utilizar sensores para coletar dados do ambiente e visualizar essas informações na plataforma multiplataforma.
  • Tópicos: Tipos de sensores (ex: luminosidade LDR), leitura analógica, envio de dados do embarcado para o software, visualização de dados (gráficos simples). Aplicação de **Estruturas de Dados** para organizar as leituras.
  • Desafio Prático: "Monitor de Luz Inteligente" - Construir um sistema onde o sensor de luz do Kit ConectaTech envia dados para a plataforma, que exibe a luminosidade atual e acende um LED se estiver escuro.
  • Recursos: Kit ConectaTech ADA (Módulo Hardware e Software), Roteiro de Aula 3, diagramas de conexão do sensor.
